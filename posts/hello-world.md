# 안녕, 세상아!

이곳은 아주 단순한 형태의 정적 블로그입니다.
프레임워크 없이 순수 HTML, CSS, JavaScript 만으로 동작하며, GitHub Pages에 바로 올릴 수 있습니다.

## 특징
- `index.html`, `styles.css`, `script.js` 세 파일이 핵심입니다.
- 마크다운 파싱은 `marked.js` CDN을 가져와서 브라우저단에서 처리합니다.
- 글을 추가할 때는 `posts/` 폴더에 `.md` 파일을 넣고 `posts.json`에 정보를 추가하면 됩니다.

너무 쉽죠? 자유롭게 꾸며서 사용해보세요!
