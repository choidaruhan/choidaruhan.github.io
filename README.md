# My Memo Web

순수 HTML/CSS/JS로 만든 메모 웹 애플리케이션입니다.

## 실행 방법

`index.html`을 브라우저에서 직접 열면 됩니다. (별도 서버 불필요)

로컬 서버를 사용하려면:

```bash
npx serve .
```

## 기능

- **포스트 목록**: 날짜순 정렬된 포스트 카드 표시
- **포스트 상세**: 카드 클릭 시 전체 내용 표시
- **검색**: 제목, 내용, 태그 기준 실시간 필터링
- **다크/라이트 모드**: 토글 버튼으로 전환 (시스템 설정 자동 감지, 선택사항 localStorage 저장)

## 프로젝트 구조

```
index.html   # 메인 페이지
style.css    # 스타일 (다크/라이트 모드, 반응형)
app.js       # 애플리케이션 로직
posts.json   # 포스트 데이터
```

## 포스트 추가

`posts.json`에 다음 형식으로 추가:

```json
{
  "id": 6,
  "title": "제목",
  "date": "2026-03-27",
  "tags": ["태그1", "태그2"],
  "content": "내용"
}
```
