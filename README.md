# 📝 My Blog

이 프로젝트는 Svelte 5와 Cloudflare 기술을 활용한 개인 블로그입니다.

## 설명

본 프로젝트는 다음과 같은 기술 스택과 원칙을 바탕으로 개발되었습니다:

- **Frontend**: Svelte 5 (Runes API 활용)
- **Deployment**: Cloudflare Pages
- **Code Principle**: 'One Function Per File' 및 명확한 명명 규칙 준수
- **Structure**: 표준화된 문서 구조를 통한 체계적인 기록

## 예시

### 로컬 개발 시작하기

```bash
npm install
npm run dev
```

### 새로운 함수 추가 시

`getUserProfile.ts` 파일을 생성하고 오직 `getUserProfile` 함수만 작성합니다.
