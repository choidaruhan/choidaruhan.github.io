# 프로젝트 개발 여정 (HISTORY)

이 문서는 프로젝트의 주요 마일스톤을 시간순으로 간결하게 기록합니다.

1. **플랫폼 이전**: Supabase에서 Cloudflare(Pages, Workers, D1)로 인프라 전면 전환
2. **기술 스택 현대화**: Vanilla JS에서 Svelte 및 TypeScript 기반 체계로 변경
3. **기능 완성**: GitHub OAuth 인증 시스템 및 게시글 CRUD 기능 구현
4. **보안 강화**: Cloudflare Access(JWT 검증) 도입으로 인증 체계 고도화
5. **구조 개편**: Core 레이어드 아키텍처(Domain, Infra, App) 도입 및 레이어 분리
6. **프레임워크 전환**: SvelteKit 및 SSG 배포 체계 마이그레이션
7. **관리 도구 구축**: 관리자 전용 로그인 및 마크다운 편집 페이지 구현
8. **문서 시스템 수립**: 원자적(Atomic) 문서 관리 및 폴더 규칙(RULES.md) 체계 도입
