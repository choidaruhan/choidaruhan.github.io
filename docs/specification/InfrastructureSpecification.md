# 인프라 구성 명세 (Infrastructure Specification)

프로젝트의 배포 및 실행 환경에 대한 명세입니다.

## 1. 프론트엔드

- **프레임워크**: SvelteKit
- **배포**: GitHub Pages (dist 폴더 배포)
- **모드**: 정적 사이트 (Static Site Generation)

## 2. 백엔드 및 DB

- **플랫폼**: Cloudflare Workers
- **데이터베이스**: Cloudflare D1

## 목적

- 프로젝트가 구동되는 전체 환경을 도식화하고 설정의 일관성을 관리합니다.
