## 🚨 진행 중인 주요 이슈

### 1. Cloudflare Access 로그인 리디렉션 루프
- **현상**: `/write` 또는 `/login` 접속 시 Cloudflare Access 인증 후 다시 로그인 페이지로 돌아오는 리디렉션 루프 발생 가능성.
- **원인 추정**: 
    - `github.io`와 `workers.dev` 간의 도메인 불일치로 인한 쿠키/세션 전송 문제.
    - CORS 설정 미흡.
- **해결 방안**: 
    - Cloudflare Access Application의 도메인 설정을 `choidaruhan.github.io/write`로 정확히 지정.
    - `handleLogin.ts`에서 리디렉션 URL 인코딩 재점검.
    - (장기적) 커스텀 서브도메인을 사용하여 TLD(Top Level Domain)를 일치시킴.
- **상태**: 🟡 분석 및 테스트 중

### 2. 마크다운 미리보기 보안 (XSS)
- **현상**: 사용자 입력 마크다운을 렌더링할 때 악성 스크립트 실행 위험.
- **해결 방안**: `dompurify`를 사용하여 `marked` 결과물을 필터링하도록 구현 예정.
- **상태**: ⏳ 대기 중
