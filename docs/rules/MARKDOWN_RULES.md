# Markdown Rules (문서화 규칙)

마크다운 문서를 작성하고 관리하는 규칙입니다.

## 1. 문서 저장 구조
- 모든 기술 명세는 `docs/` 폴더 내 주제별 서브디렉토리에 위치합니다.
- 하나의 파일은 하나의 명확한 주제만 다루어야 합니다.

## 2. 작성 가이드
- **파일명**: 대문자 스네이크 케이스(`UPPER_SNAKE_CASE.md`).
- **헤더**: 문서당 하나의 `<h1>`만 사용.
- **코드 블록**: 언어 명시 (Syntax Highlighting).
- **언어**: 한국어 기반 (기술 용어는 영문 병기).
