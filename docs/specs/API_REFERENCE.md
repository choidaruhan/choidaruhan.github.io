# API Reference (API 레퍼런스)

백엔드(Cloudflare Workers) API 엔드포인트 목록입니다.

## 게시글 (Posts)
- **`GET /api/posts`**: 목록 조회.
- **`GET /api/posts/:id`**: 상세 조회.
- **`POST /api/posts`**: 신규 작성.
- **`PUT /api/posts/:id`**: 수정.
- **`DELETE /api/posts/:id`**: 삭제.

## 인증 (Auth)
- **`GET /api/auth/me`**: 현재 세션 정보 및 인증 상태 확인.
